# TW-Skrypt
A writing plugin buillt on TiddlyWiki. Pulling inspiration from [Scrivener](https://www.literatureandlatte.com/scrivener/overview) and [Manuskript](https://www.theologeek.ch/manuskript/), coupled with my personal desire for structured writing, and my tendency to get super annoyed when a tool I'm using doesn't work quite the way I want, switch gears, and modify it (instead of writing), I started working on this. I've got local work 

## Getting Started
This entire project has been done within TiddlyWiki, running from node. All you need to do is clone this repo, checkout the `dev` branch, install tiddlywiki from npm (`npm install -g tiddlywiki` - this is global, and how I do it), and from the same directory that this file is in, run `tiddlywiki wiki --listen`, and then open up (http://127.0.0.1:8080/).

## Git branching strategy
There are a couple of different branches here. For right now (before even a beta), `main` only holds this basic ReadMe. Work and PRs should all point to the `dev` branch right now.
